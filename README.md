# n8n Projects Collection

本倉庫用來保存我在不同時期（例如 2025-08、2025-10）的 **n8n 自動化流程練習與設定**。

## 📂 專案結構

```yaml
n8n/
├─ 2025-08/
│  ├─ docker-compose.yml
│  ├─ README.md   👈 說明 8 月版本
│  └─ ...
├─ 2025-10/
│  ├─ docker-compose.yml
│  ├─ README.md   👈 說明 10 月版本
│  └─ ...
├─ .gitignore
└─ README.md      👈 專案總說明
```

- `2025-08/`  
  2025 年 8 月版本的 n8n 設定與實作。
- `2025-10/`  
  2025 年 10 月版本的 n8n 設定與實作。

> 每個資料夾都有獨立的 `docker-compose.yml` 以及對應的 `README.md`，方便快速啟動或回顧。

## 🚀 使用方式

1. 先進入想要的版本資料夾，例如：
   ```bash
   cd 2025-10
   ```
